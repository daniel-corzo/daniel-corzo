## Recent work

**[TravelHub](https://github.com/daniel-corzo/misw-4501-proyecto-de-grado)** Hotel reservations platform for 6 Latin American countries, built as a monolith-to-microservices migration on AWS. Master's thesis project, in active development.

Full production infrastructure designed and deployed before any business logic was written: 9 Terraform stacks, 11 modules, ~60 AWS resources, blue/green CI/CD for 8 microservices. Architecture validated through controlled experiments. Redis Cache-Aside cut search latency 72% (1,197ms to 327ms), clearing the 800ms p95 SLA; zero-downtime scaling to 3,600 concurrent users confirmed after diagnosing and resolving cold-start failures with multilevel health checks.
